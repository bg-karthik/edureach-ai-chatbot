# 🎓 EduReach — AI College Chatbot & Voice Agent

EduReach is an AI-powered college assistance platform that helps students with admissions, course details, placements, and counseling through both intelligent chat and voice interactions.

---

## 🚀 Live Demo

### Frontend
https://edureach-ai-chatbot-dusky.vercel.app

### Backend API
https://edureach-ai-chatbot.onrender.com

---

## ✨ Features

- 💬 AI-powered chatbot using RAG architecture
- 📞 AI voice call assistant using Vapi
- 🔐 Secure JWT authentication
- 📚 College knowledge base integration
- ⚡ Real-time AI responses
- 🎯 Personalized counseling support
- ☁️ Cloud deployment with Vercel + Render

---
